4.5 Refactoring for Efficiency and Legibility

In uniqueMath.js, you’ll see a function called mystery. Refactor the code in the function to include only one line that returns a value.

uniqueMath.js

function mystery(x, y) {
  var a = 4 * x * y;
  var b = 3 * y + 5;
  var c = a + b;
  return c;
}

Answer:

uniqueMath.js

function mystery(x, y) {
  return (4 * x * y) + (3 * y + 5);
}
